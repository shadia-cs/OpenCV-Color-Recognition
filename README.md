# OpenCV Color Recognition

## Project Overview
This project uses Python and OpenCV to recognize colors in real time using a webcam.

The program detects green, blue, and red objects, draws a bounding box around each detected object, and displays the color name.

## Features
- Real-time webcam processing
- Green, blue, and red color detection
- Bounding boxes around detected objects
- Color labels
- Press `q` to close the program

## Technologies Used
- Python
- OpenCV
- NumPy
- Anaconda
- Visual Studio Code

## Installation
```bash
pip install opencv-python numpy
```

## How to Run
```bash
python color_recognition.py
```

Show a green, blue, or red object in front of the webcam.

Press `q` to exit.
